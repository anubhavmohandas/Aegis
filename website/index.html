PRIVACY

      
Your machine's story never leaves your machine.

      

        Events, evidence, and your Memory stay in a local database on your disk. Severity is scored
        locally. Threat intelligence and AI explanations are opt-in — they only run once you add
        your own API key, and they send event metadata and file hashes, never file contents. Aegis
        records system events, not what you type, read, or look at.
