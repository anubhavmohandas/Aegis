FAQ

The questions worth asking.

Does Aegis replace antivirus?
No — on purpose. Aegis has no signatures, no verdicts, and never blocks or removes anything. It's an awareness layer that runs happily alongside your antivirus. If you need a detector, run one — and let Aegis remember what happened around it.
Does Aegis upload my files?
No. Threat-intelligence lookups send a file's hash to VirusTotal — never the file itself — and only if you enable enrichment. Everything else stays in a local SQLite database.
Does it work offline?
Mostly, yes. Monitoring, severity scoring, Memory, and evidence capture are fully local. Offline you lose VirusTotal enrichment and cloud AI explanations — unless you run Ollama locally, in which case you lose almost nothing.
Which AI models can it use?
OpenAI, Anthropic Claude, NVIDIA-hosted models, and local models via Ollama — plus any OpenAI-compatible endpoint. The provider and key are set from Settings and encrypted at rest. No key? Aegis still watches, scores, and remembers; you just lose the plain-English explanations.
What does "alpha" mean here?
Versioning tracks validation, not features. Alpha means the features are built and macOS is validated on real hardware; Windows runs validated from source with a packaged build pending, and Linux is implemented but not yet hardware-validated. The full verification log is in ARCHITECTURE.md.
Is my data really mine?
Yes. One SQLite file on your disk, exportable to JSON, CSV, or PDF from the dashboard. Delete the file and the history is gone — Aegis keeps no copy anywhere else.
DOWNLOAD

Running in three commands.

desktop_app.py is the real app — one window, live Memory, and Settings. It runs fine without an API key; you just lose the AI explanations, not the watching.

  • First run: sign in with admin / admin, then change it from Settings → Account.
  • API key lives in Settings → AI Explainer — encrypted at rest, survives self-updates.
  • Threat intelligence is opt-in: add a VirusTotal key in Settings to enable it.
  • Windows: run from an Administrator terminal for full process-monitoring power.
  • Headless box? python main.py runs the same monitors as a tray app, configured via config/config.yaml.
